# Strapi GuitarLA
  
### 🚀 Quick start

#### 1. Clonar el repositorio

```bash
git clone https://github.com/larturi/strapi-guitarla
```

#### 2. Instalar dependencias

```bash
npm install
```

#### 3. Crear el .env en base a .env.template

#### 4. Correr aplicación localmente

```bash
npm run develop
```
